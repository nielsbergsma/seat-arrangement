SeatArrangementRegistered :: forall (sc :: Nat). SeatArrangement sc 'Registered

Start of lifecycle, when a flight is scheduled, accepts reservations

SeatArrangementProposable :: forall (sc :: Nat). Reservations sc -> SeatArrangement sc 'Proposable

Accepts more reservations and removals

SeatArrangementSolicitable :: forall (sc :: Nat). Reservations sc -> SeatArrangement sc 'Solicitable

Reservations no longer accepted, collecting proposals

Assignment proposals available, accepting confirmations

End of lifecycle
